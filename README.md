### What we're building

IOTECHS is a **DePIN + AI infrastructure layer** on Solana that transforms raw IoT device data into on-chain intelligence.

We don't build another L1. We build **on top of Solana** — leveraging its speed, ecosystem, and composability.

```
Physical Devices → IotechsNet → IotechsMind → On-chain Actions
     (IoT)        (Indexing)      (AI Layer)    (Solana Programs)
```

### Core Infrastructure

| Layer | Description |
|-------|-------------|
| **IotechsNet** | Indexing & settlement layer for machine-to-machine transactions |
| **IotechsID** | Decentralized device identity — ZK proofs, on-chain NFT, reputation |
| **IotechsMind** | AI layer that converts raw signals into actionable intelligence |
| **$THINK** | SPL utility token — staking, governance, device registration, revenue share |
